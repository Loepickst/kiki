Step 0 郑重语 & 美化语 (基础礼貌)

【郑重语】定义：

不特别区分对方地位高低，为了保持“得体、礼貌”而使用的基础表达。即：在一般场合把话说得稳妥、礼貌。

1. 特殊郑重语

在服务业、商务接待、正式演讲中，通常会将「です/ます」替换成一些特殊的表达。

原词 升级版 💬 经典语境例句
ある (有/在) ございます 店员 トイレはあちらにございます。
(洗手间在那边。)
です (是) でございます 电话 はい、担当の田中でございます。
(您好，我是负责此项工作的田中。)
いい (好) よろしい 确认 ご注文は以上でよろしいでしょうか。
(您点的餐就是这些，没问题吧？)
どう (怎么样) いかが 推荐 食後のコーヒーはいかがですか。
(饭后来杯咖啡怎么样？)
今日 (今天) 本日 广播 本日はご来店ありがとうございます。
(感谢您今天的莅临。)

2. 美化语

为了让表达更文雅、柔和而添加前缀。
（它不一定是在抬高对方，更多是把说法调整得更得体。）

饮食类
  • お酒 (酒)
  • お米 (米)
  • お水 (水)
  • ご飯 (饭)
生活类
  • お風呂 (洗澡)
  • お部屋 (房间)
  • お化粧 (化妆)
  • お店 (商店)
常见固定说法

这些词里的「お」多半已经固定，学习时按整体记忆即可。

お腹 (肚子) おかず (菜肴) おやつ (零食) おにぎり (饭团)

Step 1 自谦语 (压低自己)

【自谦语】定义：

动作主语是“我自己”或“己方人员”。通过压低自己的动作，来衬托对方的地位与立场。

先看动作是否直接面向对方，再判断用哪一类。

1. 涉及对方的自谦 一般有对象

逻辑： 动作直接面向对方，或结果落实到对方身上时使用。

お / ご + 第一连用形 + する / いたす

[持有] 持つ → お持ちする

重そうですね、お持ちします。

(看起来很重呢，我来帮您拿。 → 动作指向：您)

2. 体现自我谦逊 无需对象

逻辑： 动作不一定作用于对方，重点是礼貌地表述己方行为或状态。

特殊动词 / (汉语词) + いたす

[名字] 言う → 申す

田中と申します。

(我叫田中。 → 不涉及对方，但向听众表谦虚)

Step 2 尊他语 (抬高对方)

【尊他语】定义：

动作主语是“听话人”或“长辈/上级”。通过抬高这个动作，来表示对对方的尊重。

常见尊他句式

① お + 第一连用形 + になる 最标准/最正式

多用于正式说明、公告或对长辈的稳妥表达。

[回家] 帰る → お帰りになる

先生はもうお帰りになりました。

(老师已经回去了。)

② 【被动态】 (れる / られる) 最常用/较轻松

形式与被动态相同，但语义不等于普通被动；判断时仍要看是否在抬高对方动作。

[读] 読む → 読まれる

社長は毎朝、新聞を読まれます。

(社长每天早上都会读报纸。)

③ お + 第一连用形 + です 状态描述

常用于确认持有、在场或持续状态。

[持有] 持つ → お持ちです

チケットはお持ちですか。

(您带着票吗？)

④ (汉语词) + なさる する的升级版

多接汉语词或「する」相关表达，语感郑重。

[担心] 心配する → 心配なさる

ご心配なさらないでください。

(请您不要担心。)

Step 3 特定句式表达

这一节直接识别固定结构，不再从主体出发逐步判断，适合考试和书面表达快速套用。

1. 描述自身行为 (郑重陈述) 自谦 书面常用

当说话人向听话人陈述自己的情感、或某种自我行为时，使用「申し上げます」替代普通的「します/言います」，以表达郑重的语气。

お/ご + 名词 + 申し上げます

例：心よりお詫び申し上げます。

(由衷地表示歉意。)

例：厚くお礼申し上げます。

(致以深切的谢意。)

2. 请求允许自身行为 (【使役态】自谦) 自谦 商务高频

以使役和授受组合表达“请允许我……”或“请容我……”，是商务场景的高频格式。

【使役态】(さ)せて + いただけませんか

(询问对方是否允许)

本日、早退させていただけませんか。

(今天能允许我早退吗？)

【使役态】(さ)せて + いただきます

(宣告自己将做某事)

説明させていただきます。

(那么，请允许我说明一下。)

3. 描述他人行为 (书面语中顿) 尊他 考试识别

お/ご + (第一连用形) + いただき、...

(承蒙您做某事，我方获益)

ご検討いただき、ありがとうございます。

(承蒙您探讨，非常感谢。)

お/ご + (第一连用形) + くださり、...

(您特意做某事，我方感激)

遠方よりお越しくださり、感謝いたします。

(您特意远道而来，不胜感激。)

4. 请求他人行为 (委托请求) 尊他 高频请求

根据“强制性”与“委婉度”的不同，从强到弱依次为：

お/ご～ください (指示/命令)
お/ご～願います (愿望/公文)
お/ご～いただけませんか (商量/询问)
お/ご～いただければと思います (最委婉：期望/建议)

ご検討いただければと思います。

(如果您能探讨一下那就太好了。)
